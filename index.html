FAQ

What is Auth Master?
Auth-Master is an ambitious initiative aiming to provide with a proper solution to the unbearable difficulty many users encounter while trying to access password-protected websites. The current password system in in a built in conflict, clashing between user comfort and the growing need of security (due to an increasing number of threats such as frauds, identity theft and sabotage) cry out for a different solution to enable users an easy, secure and reliable way to access services and data online. The vision of the Auth-Master initiative is to promote a gradual shift to a new, simple, easy and comfortable authentication method, without compromising the level of security and providing maximum reliability and protection for websites, users and their sensitive data.
